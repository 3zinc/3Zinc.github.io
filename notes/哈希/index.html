    哈希

    哈希简介¶

    哈希指通过一个函数 \(f(x)\) 将 \(x\) 映射到一个值上，这个值能够更加简便的进行操作。

    对于一个哈希函数，若满足在适用范围内 \(\forall x\neq y,f(x)\neq f(y)\)，那么称这个哈希函数是完美的。如果产生了不同值映射到同一哈希值上，则称为冲突。我们在设计哈希函数时要尽量避免或化简冲突。

    BKDRHash：进制哈希¶

    简介¶

    BKDRHash 是一类编码简单且实用、冲突概率小的哈希函数，一般用于字符串处理。

    如其名，其大概思想为将字符串是做一个 \(P\) 进制的数字，这个数字就是这个字符串的哈希值。由于这个值很大，一般会模上一个大的质数，但是这样会产生冲突。

    小贴士

    在程序实现中，一般使用 unsigned long long 来存储哈希值，运用自然溢出，相当于对 \(2^{64}\) 取模。避免了模运算的低效。

    实现¶

    typedef unsigned long long ull;
char s[1000005];
const ull P=131;

int len=strlen(s); ull hash;
for(int i=0;i<len;i++) hash=hash*P+s[i];

    哈希判断同构¶

    同构：指两字符串长度相同，对于 \(s1_i=s2_i\)，有 \(\forall s1_p=s1_i,s1_p=s2_p\)。

    法一：¶

    我们发现对于同构的字符串，每个字符出现位置相同。

    比如对于字符串 \(\text{CCF}\)，可用 01 串表示 \(\text{C}\) 的位置：\(\text{110}\)。

    我们可以求出所有的 01 串然后进行配对，运用 Hash 时间复杂度为 \(\mathcal{O}(|S|)\)。

    //  判断小写字符串同构 
typedef unsigned long long ull;
bool f(string s1,string s2) {
    if(s1.size()!=s2.size()) return false;
    ull hash[26];
    for(int i=0;i<26;i++) hash[i]=0;
    for(int i=0;s1[i];i++) {
        for(int j=0;j<26;j++) hash[j]<<=1;
        hash[s1[i]-'a']++;
    }
    map<ull> mp;
    for(int i=0;i<26;i++) {
        mp[hash[i]]=1; hash[i]=0;
    }
    for(int i=0;s2[i];i++) {
        for(int j=0;j<26;j++) hash[j]<<=1;
        hash[s2[i]-'a']++;
    }
    for(int i=0;i<26;i++)
        if(!mp[hash[i]]) return false;
    return true;
}

    法二¶

    我们可以通过巧妙地方式去掉字符的不同：

    对于 \(s_i\)，我们另 \(a_i\) 为 \(i-j\)，其中 \(s_j\) 为上一个与 \(s_i\) 相同的字符。特别的，如果不存在合法的 \(j\)，\(a_j=0\)。

    通过这样的方式，我们发现：同构的字符串得出的 \(a\) 数组是相同的，且仅有同构的字符串得出的 \(a\) 数组是相同的。

    //  判断小写字符串同构 
typedef unsigned long long ull;
bool f(string s1,string s2) {
    if(s1.size()!=s2.size()) return true;
    int lst[26][2];
    for(int i=0;i<26;i++) lst[i][0]=lst[i][1]=-1;
    for(int i=0;s1[i];i++) {
        int k1,k2;
        if(lst[s1[i]-'a'][0]<0) k1=0;
        else k1=i-lst[s1[i]-'a'][0];
        if(lst[s2[i]-'a'][0]<0) k2=0;
        else k2=i-lst[s2[i]-'a'][0];
        if(k1!=k2) return false;
        lst[s1[i]-'a'][0]=lst[s2[i]-'a'][1]=i;
    }
    return true;
}

    如果判断多个字符串同构，可以用哈希记录数组优化。

    二维哈希¶

    struct Hash2D {
    static const LL px= 131, py = 233, MOD =998244353;
    static LL pwx[N], pwy[N];
    int a[N] [N];
    LL hv [N] [N];
    static void init_xp() {
        pwx [0] = pwy [0] = 1;
        FOR (1, 1, N) {
            pwx[i] = pwx[i-1] * px% MOD;
            pwy[i] = pwy[i-1] * py % MOD;
        }
    }
    void init_hash(int n, int m) {
        FOR (i, 1, n +1) {
            LL s = 0;
            FOR (j, 1, m + 1) {
                s = (s * py + a[i][j]) * MOD;
                hv[i][j] = (hv[i-1][j] * px + s) % MOD;
            }
        }
    }
    LL h(int x, int y, int dx, int dy) {
        --x; --y;
        LL ret = hv[x + dx] [y + dy] + hv[x] [y] * pwx [dx] * MOD * pwy [dy]
            - hv[x] [y + dy] * pwx[dx] - hv[x + dx] [y] * pwy [dy];
        return (ret * MOD + MOD) * MOD;
    }
} ha, hb;
LL Hash2D::pwx[N], Hash2D::pwy[N];
